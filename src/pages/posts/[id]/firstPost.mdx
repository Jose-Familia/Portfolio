# Tipos de Datos en JavaScript

JavaScript es un lenguaje de programación dinámico que permite trabajar con diferentes tipos de datos. Entender estos tipos es crucial para escribir código eficiente y libre de errores. En este post, exploraremos los tipos de datos en JavaScript.

## Tipos de Datos Primitivos

### String

Una cadena de texto. Se define usando comillas simples, dobles o backticks.

```javascript
let nombre = "Juan";
let saludo = 'Hola';
let mensaje = `Hola, ${nombre}`;

Number
Representa tanto números enteros como de punto flotante.

let entero = 42;
let decimal = 3.14;
```
